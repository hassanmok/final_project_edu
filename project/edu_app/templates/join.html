Hello, World

The phrase Hello, world! is a special phrase in programming. In 1972 Brian

Kernighan, a computer scientist, wrote a tutorial for the B programming

language using this phrase. He later helped create the C programming language.

For most programmers, writing a program that prints Hello, world! is the first program they write when learning code.

Challenge

To follow this tradition, we will start with the "Hello, World!" program. On the

right, you will see a code editor with the

following code:

print("")

Inside of the double quotes, type out the

following text:

Hello, world!

After you're done, click the Submit button.

Hint: Capitalization and punctuation matter! Make

sure to type the phrase exactly as shown.
